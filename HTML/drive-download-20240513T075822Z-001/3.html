Косметика


Обзор видов косметики

    Bсе виды косметики относят к двум основным типам:

  • ухаживающая;
  • декоративная
  • иногда препараты, имеющие не только функцию ухода, но и лечебное действие, относят к отдельной категории, которую называют космецевтикой. Продаются такие товары в аптеках

    По способу производства косметические средства могут быть:

  • фабричными;
  • сделанными вручную

    Все виды косметики могут производиться с расчетом на разные категории покупателей. Есть средства:

  • для рядового потребителя;
  • для профессионального использования

Косметика для обычного потребителя представлена товарами в разном ценовом диапазоне. Обычно выделяют такие категории, как:

  • экономкласс;
  • мидл-маркет (средний ценовой уровень);
  • люкс.

      Возможна классификация косметической продукции по способу ее реализации. Продажи могут осуществляться через:

    • аптеки;
    • специализированные магазины, отделы и другие торговые точки;
    • интернет-магазины;
    • прямые продажи, то есть МЛМ или сетевой маркетинг;
    • универсальные магазины

      Можно также классифицировать косметические средства по наличию или отсутствию в них каких-либо ингредиентов. Такое деление будет основываться на той информации, что дают о своих косметических продуктах производители. Например, косметика бывает:

    • натуральная (содержащая натуральные компоненты);
    • минеральная (включающая только природные минералы);
    • органическая (только со специально отобранными компонентами растительного происхождения);
    • гипоаллергенная (не содержащая ингредиентов, провоцирующих аллергические реакции);
    • с гиалуроновой кислотой;
    • с минералами Мертвого моря;
    • с фитогормонами;
    • аюрведическая (состоит только из компонентов, перечисленных в рецептах аюрведы);
    • кислородная (содержит вещества, способные доставлять кислород в кожу);

      Есть продукты, созданные для определенных групп людей. Это может быть косметика:

    • для детей;
    • для мужчин;
    • для женщин;
    • для беременных.

      Ухаживающая косметика бывает:

    • для тела;
    • для лица;
    • для лица и тела;
    • для век;
    • для губ;
    • для рук;
    • для кутикулы;
    • для шеи и зоны декольте;
    • для волос;
    • для бровей и ресниц;
    • для ног;
    • и так далее.
